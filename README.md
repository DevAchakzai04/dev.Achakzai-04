# Achakzai 04 — Professional Portfolio

**Achakzai 04** is a modern, responsive portfolio website for a professional Web Developer & Mobile/Software Repair Expert. The site showcases services, repair expertise, recent work, and provides a contact form with social media integration.

---

## 🚀 Live Preview

Open `index.html` in any modern browser.

---

## 📁 Project Structure

> No external dependencies or build steps required.

---

## ✨ Features

- **Fully Responsive** — Works flawlessly on desktops, tablets, and mobile devices.
- **Modern Glassmorphism Design** — Frosted glass effects with animated background shapes.
- **Smooth Scrolling** — Internal anchor links with smooth navigation.
- **Services Section** — Web Development, Mobile Repair, Software Repair.
- **Repair Highlight** — Detailed list of repair services with warranty mention.
- **Portfolio Grid** — Three categories: Web Developer, Mobile Repair, Software Repair.
- **Contact Form** — Simulated form submission with validation and feedback.
- **Social Links** — Facebook, YouTube, Instagram, WhatsApp (all open in new tabs).
- **Sticky Header** — Navigation bar remains visible while scrolling.
- **No External Dependencies** — Pure HTML/CSS/JS, Font Awesome icons via CDN.
